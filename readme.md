Дошка оголошень групи — Лабораторна робота №4
Проєкт являє собою вебдодаток для розміщення оголошень, реалізований за архітектурою Client-Server.

Інструменти: npx tsc (компіляція), live-server (локальний сервер).

Структура проєкту
/backend — серверна частина на Node.js.

/frontend — клієнтська частина.

/src — вихідний код на TypeScript.

/public — скомпільовані файли, стилі та HTML.

Як запустити проєкт
1. Запуск бекенду
Перейдіть у папку сервера та запустіть його:

Bash
cd backend
node server.js
Сервер працюватиме на http://localhost:3000.

2. Компіляція TypeScript
У новому терміналі перейдіть у папку фронтенду та скомпілюйте скрипти:

Bash
cd frontend
npx tsc

3. Запуск фронтенду
Запустіть локальний сервер для відображення інтерфейсу:

Bash
npx live-server public

Функціонал
CRUD: Можливість додавання оголошень через форму та їх видалення.

API Integration: Взаємодія з сервером через fetch та власний apiClient.

Sorting: Клієнтське сортування оголошень за заголовком, категорією або автором (по кліку на заголовок таблиці).

Validation: Валідація полів на стороні сервера (400 Bad Request) та відображення помилок у формі.
